The	  risk	  assessment	  and	  risk	  and	  needs	  assessment	  tool	  instructions	  for	  each	  division	  that	  uses	  
the	  tools.	  If	  you’re	  told	  its	  proprietary,	  file	  a	  public	  records	  request	  anyway.	  
Policies	  for	  using	  the	  various	  risk	  assessments.	  Pay	  close	  attention	  to	  the	  “override”	  category.	  
This	  tells	  officials	  when	  it’s	  OK	  to	  ignore	  the	  risk	  and	  needs	  assessment	  results.	  
Contracts	  for	  purchasing,	  rollout,	  training	  and	  updates	  for	  the	  tool.	  
Evaluations	  of	  the	  tool	  for	  each	  division	  that	  uses	  it.	  States	  will	  vary	  on	  how	  often	  they	  will	  do	  an	  
evaluation	  of	  how	  the	  tool	  is	  working.	  Industry	  suggests	  assessments	  should	  be	  done	  every	  few	  
years.
