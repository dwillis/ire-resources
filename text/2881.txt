Excavando en el ambiente
IRE 2006 Fort Worth
Matthew Waite
St. Petersburg Times
Cuatro consejos para ayudarlo a excavar
1. Siempre esté atento a todo tipo de datos
El tema ambiental está lleno de información y de diferentes tipos. La lista es larga y es
normalmente una sopa alfabética de acrónimos. Organismos que prueban, permiten,
administran, regulan, todos tienen sus propias bases de datos. También se puede
encontrar mucha información útil fuera del tema estrictamente ambiental como en
donaciones de campaña, records de tierras, emails, casos judiciales, etc. cases.
2. Los datos estadísticos te pueden ayudar hoy y mañana.
Pasamos un año peleando el permiso para una base datos que resultó ser inútil para
nuestro primer propósito. Pensamos que sería genial para el análisis, pero resultó ser una
herramienta de búsqueda útil para nuestra intranet. Cuando consigues una base de datos
grande como parte de una historia, puede ayudarte aún cuando la historia ya haya sido
publicada. Las bases de datos no responden solo unas cuantas preguntas. Si puedes hacer
que funcionen online, es recomendable pasar algún tiempo creando búsquedas y filtros
predeterminados en Access que te permitirá encontrar lo que quieras.
3. Busca nuevas formas de manejar y analizar los datos para este proyecto y
para tu sección en general.
Una de las herramientas mas útiles que he desarrollado para un proyecto es el uso de un
“wiki” para almacenar información. Un “wiki”-no Wikipedia, pero sí la idea detrás de
wikipedia- puede encajarse en una base de datos masiva y fácil de buscar en la que
puedes almacenar todo tipo de información desordenada. Nosotros transcribimos docenas
de entrevistas y añadimos docenas de reportes en un “wiki”. El objetivo era doble: poder
compartir la información con cada persona del proyecto y hacer que estos datos
estuvieran disponibles para una búsqueda en el futuro. Nunca sabes cuándo algo que salió
en una entrevista hoy vaya a ser útil mañana. Se puede encontrar una lista de software de
wikis buenos y malos, usando un servidor o trabajando en tu computadora casera, aquí:
http://en.wikipedia.org/wiki/List_of_wiki_software
4. No le tengas miedo a los datos estadísticos que no hayas usado antes.
Excava lo suficientemente profundo en el tema ambiental y te encontrarás con
información y datos de los que nunca has oído hablar antes. Pregunta suficientes
preguntas y te encontrarás con muchas que las agencias no pueden responder pero que
deberían. No temas hacer el análisis tu mismo. Una parte del trabajo que hemos hecho
hasta ahora, y una buena parte del trabajo que otros medios han producido, involucran a
los reporteros haciendo sus propias pruebas y sus propios análisis de datos. Pregúntate: si
no lo haces tú, quien lo hará?
